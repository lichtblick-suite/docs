# Local Data

Lichtblick enables you to load local data files for visualization.

## Supported formats

|   | File extension               |
|---|------------------------------|
| [ROS1]() | `.bag`                |
| [ROS2]() (legacy) | `.db3`       |
| [ROS2]() (Iron onward) | `.mcap` |
| [MCAP]() | `.mcap`               |
| [PX4 Ulog]() | `.ulg`            |
