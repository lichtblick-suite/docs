# Local Data

Lichtblick enables you to load local data files for visualization.

## Supported formats

|   | File extension               |
|---|------------------------------|
| [ROS1](./ros1.md) | `.bag`                |
| [ROS2](./ros2.md) (legacy) | `.db3`       |
| [ROS2](./ros2.md) (Iron onward) | `.mcap` |
| [MCAP](./mcap.md) | `.mcap`               |
| [PX4 Ulog](./px-4.md) | `.ulg`            |
